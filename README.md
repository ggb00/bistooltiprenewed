# BiS-Tooltip Renewed

**Renewed** version of BiS-Tooltip addon for **World of Warcraft: Wrath of the Lich King (3.3.5a)**\
It adds new features, fixes old ones, adds more customization and has more recent class data.

Last item data update: `2026-05-18`\
All data is originally gathered from `wowtbc.gg`, and fixed to be compatible with **3.3.5a** version, rather than "Classic".

> While some items are objectively BiS, it's worth noting that every BiS list is subjective in some way or form.

## Installation

1. Download the addon by clicking the link below:
   - [Download BiS-Tooltip Renewed](https://github.com/ggb00/bistooltiprenewed/archive/refs/heads/main.zip)
2. Unzip the downloaded archive.
3. Rename the folder inside the zip from `bistooltiprenewed-main` to `BistooltipRenewed` (folder name should be the same as `.toc` file).
4. Move the `BistooltipRenewed` folder to your World of Warcraft AddOns folder.
    - Example: `C:\World of Warcraft WotLK\Interface\AddOns`
5. If you have WoW open - restart the game for the addon to load in properly.
